# Commitly

A MERN-based team accountability platform that helps leaders assign tasks, track progress, verify work through proof submissions, monitor team contributions, and gather anonymous feedback. Designed to improve transparency, collaboration, and responsibility in project teams, hackathons, and student groups.

## Problem It Solves

In many project teams, it is difficult to track contributions, verify completed work, and ensure accountability. Commitly provides a transparent system where tasks, deadlines, and contributions are visible and measurable.

## Features

- User Authentication (JWT)
- Team Creation and Management
- Task Assignment and Tracking
- Proof-Based Task Verification
- Progress Dashboard
- Contribution Analytics
- Anonymous Leader Feedback
- Teammate Review
- Deadline Management
- Activity History
